浅谈Arrays.asList()方法的使用
（1）该方法不适用于基本数据类型（byte,short,int,long,float,double,boolean）
（2）该方法将数组与列表链接起来，当更新其中之一时，另一个自动更新
（3）不支持add和remove方法
